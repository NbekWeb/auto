# Manrope Font Setup

## Web Font (Recommended for development)
Add this link to your web/index.html file:
```html
<link href="https://fonts.googleapis.com/css2?family=Manrope:wght@200..800&display=swap" rel="stylesheet">
```

## Local Font Files
For production and offline use, download the following files from Google Fonts:
- Manrope-Regular.ttf
- Manrope-Medium.ttf  
- Manrope-SemiBold.ttf
- Manrope-Bold.ttf

Place them in this directory and update pubspec.yaml accordingly.
